# Checkin Web Application

This a MERN Stack Web Application

# Set up locally

# Run Backend Server

    $ npm install //install server dependencies
    $ npm run server //to run with nodemon

# Run Client Server

    $ cd client
    $ npm install //install client dependencies
    $ npm start
